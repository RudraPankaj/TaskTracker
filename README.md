# Task Tracker (Vite + React + Tailwind)

Simple task tracker with:
- React (Vite)
- Tailwind CSS (dark navy UI)
- Drag-and-drop reordering (react-beautiful-dnd)
- Due-date reminders: overdue highlighting and due-soon badge
- Left navigation panel and animated Add New Task form

How to run:
1. unzip the project
2. npm install
3. npm run dev
